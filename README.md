# Portfolio

This is a portfolio project built using the following tech stack:

- React 
- Typescript
- Sass

## Installation

To install the dependencies for this project, run the following command:

```bash
yarn install
```

## Run

To run the project, use the following command:

```bash
yarn start
```

## Deploy to GitHub Pages

The project is set up to deploy to GitHub Pages via GitHub Actions. Use a **user/org site** repo named `username.github.io` so the site is served at root (e.g. `https://username.github.io`).

1. In your repo on GitHub: **Settings → Pages → Build and deployment** → set **Source** to **GitHub Actions**.
2. Push to `main` (or `master`); the workflow will build and deploy.
